# Petunjuk Penggunaan

## 1. Setup Template Icon
1. Ambil **screenshot** semua jenis icon cuaca dari layar game.
2. Hapus **background** dari setiap icon tersebut.
3. Ganti semua icon cuaca yang ada di folder `icons` dengan icon hasil screenshot tadi.

---

## 2. Setup Audio Alert
1. Ganti semua file alarm yang ada di folder `assets/audio`.

Keterangan nama file:
- `1m.mp3`  → Alarm untuk **1 menit**
- `3m.mp3`  → Alarm untuk **3 menit**
- `5m.mp3`  → Alarm untuk **5 menit**
- `30s.mp3` → Alarm untuk **30 detik**
- `kemarau.mp3` → Peringatan jika terjadi **perpanjangan jeda cuaca**
- `warning.mp3` → Peringatan detik-detik kadang **cuaca tiba-tiba muncul saat masa perpanjangan**

---

## 3. Pengaturan Posisi Radar
1. Gunakan tombol **TOGGLE MODE** untuk mengganti fungsi kontrol navigasi.  
   Keterangan fungsi yang aktif akan ditampilkan di bawah tombol **TOGGLE MODE**.

2. Gunakan mode berikut untuk mengatur radar:
   - **MOVE** → Menggeser posisi radar
   - **RESIZE** → Mengatur ukuran / zoom radar

   Atur posisi dan ukuran radar sampai **jenis cuaca dapat terdeteksi dengan benar**.

3. **Catatan:**  
   Berhati-hatilah saat menggunakan **Calibration Mode**.  
   Fitur ini masih dalam tahap pengembangan dan proses resize gambar secara realtime cukup berat.
